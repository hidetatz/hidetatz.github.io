2024年11月25日 変えたい過去

アイナ・ジ・エンドという歌手の人がいて、まあいることは前から知っていたんだが、最近「名前かっこいいな」とふと思った。 うちで飼ってる猫は「よもぎ」という名前なんだけど、「ヨモギ・ジ・エンド」に改名したい。

↑ヨモギ・ジ・エンド

タイムマシンで過去に戻れるなら、いくつか戻って歴史を変えたいことがある:

  • アラビア数字を考えてる会議に参加して、「6と9はひっくり返すと同じになるからどっちか形変えたほうがいいのではないか」と言いたい
  • JSONの仕様を決めた人のところに行って、「コメントを書けるようにしたほうがいいのではないか」と言いたい
  • アセンブラを考えた人のところに行って、「Intel記法とAT&T記法でオペランドの順番を逆にするのはややこしいからやめたほうがいい」と言いたい
  • cdコマンドを作ってる人のところに行って、「引数が与えられなければ、ホームディレクトリに移動するという挙動はおかしいと思う」と言いたい
  • vimを作ってるところに行って、「jumpoptionsのデフォルトはstackの挙動のほうがいい気がする」と言いたい
